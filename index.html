Hello World!

Cascading Style Sheet atau CSS adalah bahasa pemrograman berbasis aturan yang dikembangkan oleh World Wide Web Consortation (W3C) pada tahun 1996. CSS dikembangkan karena HTML yang berfungsi untuk memformat halaman web tidak bisa menambahkan gaya atau letak tingkat lanjut.


Misalnya, pada HTML versi 3.2 sudah diperkenalkan tag agar gaya teks bisa diubah. Namun, web designer amatir menyelahgunakannya dan tidak ada opsi fallback seperti yang sekarang tersedia di CSS. Jadi, pilihan terbaiknya adalah tetap memisahkan pemformatan (HTML) dan penataan gaya (CSS)
